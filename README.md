# Ashenafi Habte Portfolio

Personal portfolio website for Ashenafi Habte Atlaw.

## About

This project is a modern personal portfolio website created using HTML, CSS, and JavaScript.  
It shows my skills, projects, education, and contact information.

## Features

- Responsive design
- Dark and light mode
- Animated sections
- Typing text effect
- Contact form
- Download CV button
- GitHub and LinkedIn links
- Secret gallery feature
- Multi-language support (English and Amharic)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- Typed.js
- Remix Icons

## Projects Included

### Housing Allocation Portal
University housing allocation system using PostgreSQL and Tailwind CSS.

### File Uploading System
Secure file upload and management system using PHP.

### Online Shopping System
Java desktop shopping application with MySQL database.
